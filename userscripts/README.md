# Userscripts

Userscripts (Tampermonkey / Violentmonkey / OrangeMonkey) que rodam em cima
do tracker. Independentes do tema CSS — instalação separada.

## Scripts

| Arquivo | O que faz |
|--|--|
| [`capyppuccin-imgbb.user.js`](capyppuccin-imgbb.user.js) | Upload de imagem no chat (paste + botão) via ImgBB. Protótipo do que vai virar feature nativa. |

## Como instalar

1. Tampermonkey / Violentmonkey / OrangeMonkey no navegador
2. Abre o `.user.js` no GitHub raw → extensão detecta e oferece instalar
3. Hard refresh em `capybarabr.com`
